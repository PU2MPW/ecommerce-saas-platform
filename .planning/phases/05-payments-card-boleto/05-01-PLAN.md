# Plan 05-01: Card & Boleto Integration

## Objetivo
Integrar cartão de crédito (parcelamento) e boleto bancário via Mercado Pago (simulado).

## Tasks

### Task 1: Payment Library
Criar `src/lib/payment.ts` com funções para processar pagamentos (simulação).

### Task 2: Card Payment Form
Criar `src/components/checkout/CardPaymentForm.tsx` com campos de cartão e parcelas.

### Task 3: Card API
Criar `src/app/api/payment/card/route.ts` para processar cartão.

### Task 4: Boleto API
Criar `src/app/api/payment/boleto/route.ts` para gerar boleto.

### Task 5: Webhook
Criar `src/app/api/webhooks/payment/route.ts` para confirmar pagamentos.

### Task 6: Payment Pages
Criar páginas de pagamento para cartão e boleto.

- [ ] Pagamento com cartão de crédito funciona
- [ ] Parcelamento em até 12x disponível
- [ ] Boleto bancário gerado
- [ ] Webhook processa confirmação
- [ ] Status do pedido atualizado
